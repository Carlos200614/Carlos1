# Carlos1

Portfólio em estilo Minecraft com fundo animado, botões de menu pixelados, inventário interativo, seção para animações do Blockbench e galeria de projetos.

## Publicação no GitHub Pages

1. Faça o push do projeto para o GitHub.
2. Acesse as configurações do repositório.
3. Em Pages, selecione a branch principal como fonte.
4. O site ficará disponível em uma URL do formato: https://seu-usuario.github.io/Carlos1/

O projeto está preparado para funcionar diretamente como uma página estática sem necessidade de build.
